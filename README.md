### 简介
企业微信2.8.7新版本上线了“群机器人”功能，
wx-chatbot是基于企业微信群机器人API的脚手架。
使用wx-chatbot 快速接入 群机器人。
下载直接可以使用。
src/test/java/com/yanceyzhang/chatbot/demo/ 下面有测试类
####  测试请将TestConfig 下的 webhook 地址修改为自己的机器人地址
